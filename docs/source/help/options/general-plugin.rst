General plugin

Autorefresh
-----------

Set autorefresh for every available plugin.

   <form action="">
    Example:<br>
    <ul>
    <li>Account <input type="checkbox" name="account-autorefresh-on" checked> &mdash; autorefresh is turned on.<br>
    <li>Account <input type="checkbox" name="account-autorefresh-off"> &mdash; autorefresh us turned off.<br>
    </ul>
   </form>
