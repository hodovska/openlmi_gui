Toolbar
=======

Add new system
--------------

Prompt user for ip/domain name of desired system.

Delete system
-------------

Delete selected system.

Discover systems
----------------

Discover systems on network using SLP.

Refresh
-------

Refresh tab of selected system.

Save lmishell code
------------------

Save lmishell code into user defined file.

Apply changes
-------------

Apply changes made in tab.

Power management
----------------

Perform action on selected system. |br|
Possible actions: Reboot, Shutdown, Force reset, Force off, Wake on lan

LMIShell code editor
--------------------

Display dialog with lmishell code representing changes made in current tab.

Filter
------

Show filter if available in current tab.
